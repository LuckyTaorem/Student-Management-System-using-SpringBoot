To view all the student: GET: Eg: api/student
To view student by id: GET: Eg: api/student/12203018
To Add new student: POST: Header: (Content-type: application/json)

Body:

{

"studentId":"STUDENT_ID",

"firstName":"STUDENT_FIRSTNAME",

"lastName":"STUDENT_LASTNAME",

"course":"STUDENT_COURSENAME",

"cgpa":"STUDENT_CGPA",

"dob":"STUDENT_DOB"

}


Eg:

{

"studentId":"12208382",

"firstName":"Kunal",

"lastName":"Yadav",

"course":"CSE Btech",

"cgpa":"7.6",

"dob":"16/11/2003"

}
To update student details: PUT: Eg: api/student/update/12203018
Header: (Content-type: application/json)

Body:

{

"studentId":"STUDENT_ID",

"firstName":"STUDENT_FIRSTNAME",

"lastName":"STUDENT_LASTNAME",

"course":"STUDENT_COURSENAME",

"cgpa":"STUDENT_CGPA",

"dob":"STUDENT_DOB"

}


To delete student details: DELETE: Eg: api/student/delete/12203018
